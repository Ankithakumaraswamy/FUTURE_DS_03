# Student Feedback Analysis

## Project Overview
This project analyzes student feedback to assess teaching effectiveness. By using Python (Pandas/Seaborn), I identified key performance areas and sentiment trends.

## Key Insights
- **Top Strength:** Instructors are highly versed in their subjects.
- **Area for Improvement:** Pedagogical delivery and assignment support require more attention.
- **Recommendation:** Implementing modular "Micro-Surveys" can improve real-time engagement.

## Technologies Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook

## How to View
The full analysis, including code and visualizations, is in the `Student_Feedback_Analysis.ipynb` file.
